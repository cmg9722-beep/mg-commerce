[1] 앱 설치 방법
----------------------------------------

■ 아이폰
  1. Safari로 https://mg-commerce.onrender.com/ 접속
  2. 하단 공유 버튼 탭
  3. "홈 화면에 추가" 선택
  4. "추가" 누르면 완료

■ 안드로이드
  1. Chrome으로 https://mg-commerce.onrender.com/ 접속
  2. 상단에 "앱 설치" 배너 뜨면 클릭
  3. 안 뜨면 메뉴(점3개) → "앱 설치" 또는 "홈 화면에 추가"

■ PC (윈도우)
  1. Chrome으로 https://mg-commerce.onrender.com/ 접속
  2. 주소창 오른쪽에 설치 아이콘 클릭
  3. "설치" 누르면 바탕화면에 앱 생성


[2] 관리자 대시보드 사용법
----------------------------------------

접속: https://mg-commerce.onrender.com/admin
비밀번호 입력 후 로그인

■ 9개 메뉴 설명

1. 대시보드
   - D-Day 카운트다운 (쿠팡 프로모션 데드라인)
   - KPI 요약 (제품수, 공급사수, 평균마진 등)
   - 마일스톤 진행 현황
   - 전체 마진 현황

2. 제품 발굴
   - 추천 검색 카테고리 확인
   - 신규 제품 후보 등록
   - 자동 점수 평가 (마진율, 경쟁강도, KC인증 등)
   - 점수순 후보 목록 관리

3. 공급사 관리
   - 공급사 정보 (이름, 위치, 등급, 가격)
   - 타임라인 추적
   - 주문 생성 및 상태 관리

4. 마진 계산
   - 실시간 CNY→KRW 환율 자동 조회
   - 전체 제품 마진 자동 계산
   - 프로모션 적용/미적용 비교
   - 개별 시뮬레이션 (단가, 판매가, 수량 입력)

5. 파이프라인
   - 제품별 진행 단계 추적
   - 발굴 → 공급사선정 → 샘플 → 검수 → 본발주 → 쿠팡등록

6. 메시지 생성
   - 중국어 메시지 자동 생성
   - 유형: 샘플주문, 단가문의, 인증서류요청, 창고주소전달, 감사
   - 생성 후 복사 버튼 → 1688 채팅에 붙여넣기

7. 쿠팡 보조
   - 상품명 SEO 최적화 생성
   - 가격 전략 추천
   - 상품등록 체크리스트
   - 상세페이지 HTML 자동 생성

8. 검수 체크
   - 품질 검수 체크리스트
   - 제품별 검수 항목 체크

9. 고객 문의
   - 홈페이지에서 접수된 문의 확인
   - 답변 작성 및 상태 관리 (대기/답변완료/종료)
